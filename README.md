## MB coding challenge

This is a simple web app that parses some sample CSV data and uses D3 and Backbone to display some visualizations of it. 

Demo [here](https://amd-mbchallenge.herokuapp.com/).

### To run locally:

`shotgun app/app.rb`

`http://localhost:9393`

### To run tests:

`rake jasmine`

`http://localhost:8888`
